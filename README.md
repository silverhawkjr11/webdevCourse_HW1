# 5 Places I Want To Visit

A simple website showcasing five European destinations I'd love to explore.

## Features

- Interactive card-based navigation with hover effects
- Responsive button design
- Catppuccin dark theme color scheme
- Detailed pages for each destination with images and descriptions

## Destinations

1. **Vienna, Austria** - Classical music, historic architecture, and coffee culture
2. **Rome, Italy** - Ancient history, the Colosseum, and authentic Italian cuisine
3. **Athens, Greece** - Birthplace of democracy, the Acropolis, and Mediterranean culture

5. **Paris, France** - The Eiffel Tower, world-class museums, and romantic charm

## Files

- `index.html` - Homepage with navigation cards and map
- `Austria.html`, `Italy.html`, `Greece.html`, `Spain.html`, `France.html` - Individual destination pages
- `style.css` - Unified stylesheet with custom theming
- `images/` - Destination images and map

## Design

The site uses a dark Catppuccin color palette with rounded corners, smooth transitions, and hover effects throughout for a modern, polished look.
